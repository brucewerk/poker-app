📝 COMANDOS ÚTEIS PARA MANUTENÇÃO

Atualizar local:
bash
git pull origin master
npm install
npm run dev

Deploy manual na Vercel:
bash
vercel --prod

Backup do banco de dados:
bash
mongodump --uri "sua_mongodb_uri" --out ./backup
